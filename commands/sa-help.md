# /sa-help

List all available SearchAtlas toolkit commands.

## Instructions

Display this command reference to the user:

```
🧠 SearchAtlas Toolkit — Commands

🎯 DIAGNOSTICS & REPORTS
  /sa-scout               Read-only diagnostic across SEO, GBP, PPC, content, and AI visibility
                          — holistic scoring, Site Explorer, GBP audit, Report Builder report + local HTML
  /sa-business-report     Deep dive on a single business — OTTO, brand vault, Site Explorer,
                          GBP, PPC, and LLM visibility with recommendations
  /sa-my-account          Account overview — all OTTO projects, brand vaults, GBP locations,
                          PPC campaigns, content artifacts, and quota

🚀 ONBOARDING & BRAND
  /sa-onboard-client      Guided new client onboarding wizard — pulls from SearchAtlas brand vault
                          or walks through manual brand profile setup
  /sa-sync-client         Two-way sync between local brand-profile.md and SearchAtlas brand vault

🔧 MARKETING WORKFLOWS
  /sa-run-seo             SEO onboarding or monthly maintenance using OTTO and holistic audit tools
  /sa-run-gbp             GBP profile optimization, post scheduling, and review management
  /sa-run-ppc             Build and launch a PPC campaign using SearchAtlas Ads tools
  /sa-run-content         Generate articles from topical maps using SearchAtlas Content Generator
  /sa-run-pr              Create and distribute press releases + authority building via Link Lab
  /sa-run-visibility      LLM visibility audit and sentiment analysis using AI Visibility tools
  /sa-summit-shot         Run a single bounded play from the Summit playbook (19 plays available)

💬 SHARING
  /sa-send-slack          Post a report or message to Slack (supports multiple named channels)
  /sa-send-discord        Post to Discord via webhook
  /sa-send-email          Send an email via Resend API
  /sa-send-circle         Post to a Circle community space

🛠️ SETUP & UTILITIES
  /sa-setup-integrations  Configure Slack, Discord, email, and Circle webhook integrations
  /sa-security-scan       Scan any GitHub repo for threats before cloning or running it
  /sa-build-website       Build a client website using SearchAtlas Website Builder
  /sa-rebuild-website     Rebuild or redeploy an existing client website
  /sa-help                This command list
```

## Output Format

Display the command table above using monospace formatting. Keep it clean — no extra explanation needed.
