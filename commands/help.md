# /searchatlas:help

List all available SearchAtlas toolkit commands.

## Instructions

Display this command reference to the user:

```
🧠 SearchAtlas Toolkit — Commands

🎯 DIAGNOSTICS & REPORTS
  /searchatlas:scout               Read-only diagnostic across SEO, GBP, PPC, content, and AI visibility
                          — holistic scoring, Site Explorer, GBP audit, Report Builder report + local HTML
  /searchatlas:business-report     Deep dive on a single business — OTTO, brand vault, Site Explorer,
                          GBP, PPC, and LLM visibility with recommendations
  /searchatlas:my-account          Account overview — all OTTO projects, brand vaults, GBP locations,
                          PPC campaigns, content artifacts, and quota

🚀 ONBOARDING & BRAND
  /searchatlas:onboard-client      Guided new client onboarding wizard — pulls from SearchAtlas brand vault
                          or walks through manual brand profile setup
  /searchatlas:sync-client         Two-way sync between local brand-profile.md and SearchAtlas brand vault

🔧 MARKETING WORKFLOWS
  /searchatlas:run-seo             SEO onboarding or monthly maintenance using OTTO and holistic audit tools
  /searchatlas:run-gbp             GBP profile optimization, post scheduling, and review management
  /searchatlas:run-ppc             Build and launch a PPC campaign using SearchAtlas Ads tools
  /searchatlas:run-content         Generate articles from topical maps using SearchAtlas Content Generator
  /searchatlas:run-pr              Create and distribute press releases + authority building via Link Lab
  /searchatlas:run-visibility      LLM visibility audit and sentiment analysis using AI Visibility tools
  /searchatlas:summit-shot         Run a single bounded play from the Summit playbook (19 plays available)

💬 SHARING
  /searchatlas:send-slack          Post a report or message to Slack (supports multiple named channels)
  /searchatlas:send-discord        Post to Discord via webhook
  /searchatlas:send-email          Send an email via Resend API
  /searchatlas:send-circle         Post to a Circle community space

🛠️ SETUP & UTILITIES
  /searchatlas:setup-integrations  Configure Slack, Discord, email, and Circle webhook integrations
  /searchatlas:security-scan       Scan any GitHub repo for threats before cloning or running it
  /searchatlas:build-website       Build a client website using SearchAtlas Website Builder
  /searchatlas:rebuild-website     Rebuild or redeploy an existing client website
  /searchatlas:help                This command list
```

## Output Format

Display the command table above using monospace formatting. Keep it clean — no extra explanation needed.
