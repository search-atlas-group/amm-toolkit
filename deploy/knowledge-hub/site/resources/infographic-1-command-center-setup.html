One-Time Setup Checklist

Do this once — takes about 15 minutes

Step 01 · Install Warp

Download Warp Terminal

Warp is a modern, AI-powered terminal. Free to start. Go to warp.dev and download for Mac or Windows.

# After download, open Warp and sign in
# Free tier: 100 AI requests/month
# Pro: unlimited — recommended for agencies
Step 02 · Install Claude Code

Install the Claude Code CLI

Run this one command in Warp. It installs Claude Code globally so you can use it from any folder.

npm install -g @anthropic-ai/claude-code
# Then log in:
claude
# Follow the login prompt (browser opens)
Step 03 · Connect Search Atlas

Add Search Atlas MCP

This gives Claude access to all 60+ Search Atlas tools — SEO, GBP, content, reporting, and more.

claude mcp add searchatlas \
  --transport http \
  https://mcp.searchatlas.com/api/v1/mcp
# You'll be prompted for your API key
Step 04 · Add Communication Tools

Connect Slack, ClickUp & More

Claude Code has built-in connections for all major agency tools. Add them one by one or use the built-in command.

# Inside a Claude session, type:
/mcp
# Browse and add: Slack, ClickUp,
# Gmail, Google Calendar, Linear
Step 05 · Install Agency Commands

Set Up Slash Commands

The Agentic Marketing Mastermind gives you 13 pre-built slash commands for SEO, GBP, PPC, content, and reporting.

# Clone the command library:
git clone https://github.com/your-repo
# Run the setup script:
./setup.sh
Step 06 · Write Your CLAUDE.md

Tell Claude About Your Agency

Create a CLAUDE.md file in your working folder. Claude reads this at the start of every session — set it once, works forever.

# Start Claude, then type:
/init
# Add your clients, tone, preferences
# Claude remembers everything here

Checking Your Connections

Verify everything is working before you start a session

Check from the command line CLI

$claude mcp list
NAME STATUS TRANSPORT
✓ searchatlas connected http
✓ slack connected http
✓ clickup connected http
✓ gmail connected http
⚠ gcal auth needed
$claude --version
Claude Code 1.x.x ✓ up to date

Check from inside a session In-App

# Type these inside Claude Code:
>/mcp
→ Opens the MCP management panel
→ Shows all servers + green/red status
→ Click to reconnect or re-auth
>/status
→ Account, model, version info
>what MCP tools do you have access to?
→ Claude lists all available tool names

Fix a broken connection Troubleshoot

# Re-add a server (safe to run again):
$claude mcp remove searchatlas
$claude mcp add searchatlas \
--transport http \
https://mcp.searchatlas.com/api/v1/mcp
# Diagnose Claude Code itself:
$claude doctor
→ Runs full health check
→ Points to the exact issue

Test Search Atlas is working Quick Test

# Start Claude and type this:
>/my-account
→ Claude should list your Search Atlas
projects, GBP locations, campaigns
# Or ask directly:
>show me my Search Atlas projects
✓ Connected: returns your OTTO projects
✗ "credentials not provided" = check API key
✗ "Tool not found" = re-add MCP server
