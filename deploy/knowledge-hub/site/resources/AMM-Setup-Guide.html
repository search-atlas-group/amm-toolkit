Agentic Marketing Mastermind

New User Setup Guide
Setup time: ~15 minutes  ·  Mac & Windows  ·  All links are clickable
Step 1

Get a Claude Subscription

You need a Claude Pro ($20/mo) or Claude Max ($100/mo) plan to use Claude Code.

  1. Go to claude.ai
  2. Sign up or log in
  3. Click Upgrade to Pro (or Max)
  4. Complete payment
Which plan?

Pro is fine for getting started. Max gives 5x more usage — recommended if you run workflows daily for multiple clients.

Step 2

Install Warp Terminal

Warp is a modern terminal that works great with Claude Code. Free to use.

Mac

  1. Go to warp.dev
  2. Click Download for Mac
  3. Open the .dmg and drag to Applications
  4. Open Warp and sign up

Windows

  1. Go to warp.dev
  2. Click Download for Windows
  3. Run the installer
  4. Open Warp and sign up
Step 3

Install Claude Code

Open Warp and run:

npm install -g @anthropic-ai/claude-code

Verify it installed:

claude --version
Don't have npm?

Install Node.js first from nodejs.org (LTS version). This gives you both node and npm. Restart Warp after installing.

Step 4

Create Your Working Directory

This is where all your client work will live. Open Warp and run:

mkdir -p ~/agency
Tip

You can name this anything — ~/agency, ~/clients, ~/marketing — whatever makes sense to you.

Step 5

Clone the Repository

This downloads the entire marketing toolkit to your computer:

cd ~/agency
git clone https://github.com/manickbhan/agentic-mastermind.git
cd agentic-mastermind
git not found?

Mac: Run xcode-select --install in Warp
Windows: Install from git-scm.com

Step 6

Run the Setup Wizard

bash setup.sh

The wizard walks you through 3 steps:

  1. SearchAtlas API Key (required) — Go to searchatlas.com → Settings → API Keys → Copy your key. Paste it when prompted (it won't show on screen — that's normal, it's hidden for security).
  2. Communication Channels (optional) — Choose which channels you use to share results with clients:
    • Slack — needs a webhook URL (create one here)
    • Discord — needs a webhook URL (Server Settings → Integrations → Webhooks)
    • Email — needs a Resend API key (sign up free here)
    • Circle — needs an API key from your Circle community settings
    You can skip this and set them up later.
  3. MCP Server Configuration — happens automatically. The wizard connects Claude Code to SearchAtlas for you.
Step 7

Launch Claude Code

Make sure you're inside the project directory, then start Claude Code:

claude

Claude Code will start up and connect to SearchAtlas via MCP automatically.

Step 8

Set Permissions to Bypass Mode

Once Claude Code is running, type:

/permissions

Select "Bypass permissions" — this allows Claude Code to run all tools without asking for confirmation on every API call.

Why bypass?

Marketing workflows call many SearchAtlas tools in sequence. Without bypass mode, you'd have to manually approve every single API call, which makes workflows unusable. You can toggle permissions anytime with Shift+Ctrl+P (Win) or Shift+Cmd+P (Mac).

Step 9

Verify Everything Works

Test that commands are installed:

/help

You should see all 14 commands listed. Then test your SearchAtlas connection:

/my-account

This should list your businesses, projects, campaigns, and GBP locations. If you see data, you're all set.

Step 10

Onboard Your First Client

/onboard-client

This guided wizard walks you through:

  1. Creating an OTTO project for the client's domain
  2. Setting up a Brand Vault (brand voice, logos, target audience)
  3. Running an initial SEO audit
  4. Connecting their Google Business Profile (if applicable)
  5. Setting up PPC campaigns (if applicable)
