Windows users — what to expect during this phase
If a step pauses or an error appears, these are the three most common causes:

Not running as Administrator. PowerShell must be opened with "Run as Administrator" before pasting the quickstart command. If winget or npm fails, this is almost always why.

WSL prompt. Some tools may ask to install Windows Subsystem for Linux. Accept it — it's required. A reboot may follow. After rebooting, re-run the quickstart command; it will pick up where it left off.

Network error during download. Hotel or public Wi-Fi can cause SSL handshake failures that look like setup errors. If you see "exit code 35" or a curl/npm fetch failure, switch to a mobile hotspot and re-run the command.
