Power Prompts That Work

Copy, customize, and use — these are high-signal prompts that get great results

FOR SEO REPORTING
"Pull the top 20 ranking keywords for acme.com from Search Atlas, compare to last month, and write a 2-paragraph executive summary of ranking progress suitable for a non-technical client."
Why it works: Specifies the data source, scope (20 keywords), comparison period, and output format (executive summary, non-technical tone). Claude has everything it needs.
FOR LOCAL SEO / GBP
"Check the GBP profiles for all 12 locations of [client] in Search Atlas. Identify any with incomplete info or low optimization scores, then create prioritized ClickUp tasks to fix them."
Why it works: Scopes the work (12 locations), defines the filter (incomplete/low score), specifies the output (prioritized ClickUp tasks), and chains two tools together.
FOR CONTENT CREATION
"Using Content Genius, create 5 blog posts targeting the keywords in @keyword-list.txt, matching the brand voice in @brand-guidelines.md. Add them as ClickUp tasks assigned to the content team."
Why it works: References specific files with @, sets the quantity (5), specifies the tool (Content Genius), and ends with a clear action (ClickUp tasks assigned).
FOR CLIENT COMMUNICATION
"Read the last 5 emails from client@company.com, summarize the open items, draft a professional response to each, and create ClickUp tasks for any commitments we made."
Why it works: Scopes to recent emails (last 5), asks for a summary, response draft, and task creation — a full loop from inbox to action.
FOR SCHEDULING
"Find a 45-minute slot next week when both Sarah and I are free, create a 'Q4 Strategy Planning' event, invite Sarah, and add an agenda to the event description."
Why it works: Sets duration, time horizon, attendees, event name, and asks for a bonus (agenda) — all the details needed to create the calendar event without back-and-forth.
FOR AI SEARCH VISIBILITY
"Run an LLM visibility check for [client]. Show where they appear (or don't) in ChatGPT and Perplexity answers for their top 10 keywords. What should we do to improve their AI search presence?"
Why it works: Names the tools (ChatGPT, Perplexity), scopes the query (top 10 keywords), and asks for analysis AND actionable recommendations in one go.

Best Practices & What to Avoid

The habits that separate agencies getting 10× productivity from those struggling with Claude Code

🎯

Write Better Prompts

  • Be specific: name the client, the tool, the output format
  • Include "done" criteria: "confirm all 10 keywords have data"
  • Use @file to reference existing docs and briefs
  • Paste screenshots of dashboards or designs directly
  • Chain tasks: "then create a ClickUp task for each item"
🗂️

Manage Sessions Well

  • Use /clear between different clients — avoid context bleed
  • Use /rename to save sessions (e.g., "acme-q3-report")
  • Use /compact in long sessions before running out of memory
  • Use /resume to pick up cross-day work without re-explaining
  • Run one client per session for clean outputs
⚡

Work Faster with Warp

  • Save common claude commands to Warp Drive workflows
  • Use # + plain English to generate terminal commands
  • Right-click any error → "Ask Warp AI" for instant help
  • Open multiple Warp panes for parallel client sessions
  • Set up Warp workflows for your monthly reporting run

⚠️ Common Mistakes to Avoid

Mixing clients in one session
Use /clear between clients — Claude will mix up data otherwise
Vague prompts with no scope
"Update the Q3 report for Acme with the top 10 keyword rankings from this month"
Giant CLAUDE.md files
Keep it under 100 lines — long instructions are ignored. Be concise and specific.
Not checking MCP status before a big session
Run /mcp or claude mcp list before important client work to verify all tools are live
Correcting Claude over and over
If you've corrected twice, /clear and rewrite the prompt with what you learned
Approving actions without reading them
Claude asks before publishing or creating in external tools — always review the plan first
