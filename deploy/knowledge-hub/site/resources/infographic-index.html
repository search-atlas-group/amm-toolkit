◈ Best Practices
Habits That Drive 10× Productivity
🎯 Better Prompts
Name the client, tool, and output format
Include "done" criteria in every prompt
Use @file to reference documents
Chain tasks: "then create ClickUp tasks"
Paste screenshots — Claude reads images
🗂️ Session Hygiene
/rename before any real work starts
/clear between every client — no exceptions
/compact in long sessions before slowdown
/export before closing important sessions
Never mix clients in one session
⚡ Warp Habits
Save commands to Warp Drive for reuse
Type # + plain English for AI commands
Right-click errors → "Ask Warp AI"
Multiple panes for parallel clients
Never ignore MCP connection warnings
🚫 Common Mistakes
Vague prompts with no scope or client name
Pasting API keys into the chat
Approving without reading the plan
Correcting Claude 3× — /clear instead

🔧 Back on Track
Wrong or confusing answers?
/clear and rewrite with a more specific prompt. You lost nothing — just context.
A tool isn't connecting?
Type /mcp to check status. Remove and re-add the server. 30 seconds.
Session feels slow or foggy?
/compact to free memory. Still bad? Export, clear, resume fresh.
Can't find a report?
Ask Claude where it saved it. Going forward: always end with /export → /reports/.
Feeling overwhelmed?
3 things only: Open Warp → type claude → describe what you need. That's it.

You don't need to understand the code.
Just describe the outcome clearly.
