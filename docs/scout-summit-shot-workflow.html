The flow

Diagnose. Decide. Drop.

No guesswork. Run /scout on any domain — yours or a client's — to get a prioritized hit-list of what to fix first, with each item tied to a specific shot you can fire.

01 — DIAGNOSE

/scout {domain}

Read-only. Pulls every pillar in parallel from SearchAtlas. Skips cleanly when something isn't connected.

  • OTTO health + pillar scores
  • Site Explorer (kws, backlinks)
  • Brand vault completeness
  • GBP · PPC · LLM Visibility
02 — DECIDE

Action plan

Each pillar gets a status. Top 4-6 weaknesses become a numbered priority list — every item points to the exact command to run next.

  • ✅ Healthy / ⚠️ Needs work / ❌ Missing
  • SA Report Builder → client-shareable
  • Local self-contained HTML → internal record
03 — DROP

/summit-shot N

Pick one play from the menu. Atomic, bounded, drafts by default. Every run logged and followed by next-move suggestions.

  • 19 plays from Summit Days 1–5
  • Confirm before any destructive write
  • Re-run /scout to measure the lift
